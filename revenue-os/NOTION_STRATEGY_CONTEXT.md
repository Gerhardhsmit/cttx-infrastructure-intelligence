# CTTX Revenue OS — Strategy Context (for Notion)

## North Star Metric
**Client quote requests per week.**

## System Architecture

```
CRUISER Speed Test
        │
        ▼ (30 seconds)
Lead Qualification Agent (Score 0-100)
        │
        ▼
Discovery Agent (SPICED — uncover pain)
        │
        ▼
Challenger Agent (teach using their data)
        │
        ▼
Quote Preparation Agent (assemble pack)
        │
        ▼
Follow-up Agent (6-touch, 25-day sequence)
        │
        ▼
Sales Coach (analyse every outcome)
        │
        ▼
Management Dashboard (real-time metrics)
```

## Lead Qualification Scoring (Agent 1)

Scores 0-100 based on:
- Customer type: Residential / SME / Business / Enterprise
- Existing provider
- Current monthly spend
- Address / location
- Coverage availability
- Pain level (speed gap)
- Buying timeframe

Tiers:
- 85-100: HOT — immediate callback
- 70-84: WARM — same-day contact
- 50-69: NURTURE — add to sequence
- 30-49: COLD — monitor
- 0-29: DISQUALIFIED

## Discovery (Agent 2) — SPICED

Questions:
- What made you test your speed today?
- Is this affecting work?
- Is this your home or business?
- How many people rely on the connection?
- Are Teams calls dropping?
- Are CCTV cameras affected?
- Is this costing your business money?

## Challenger (Agent 3) — Teach, Don't Sell

Example:
"You are paying R2,300/month. Based on your speed test, you're only receiving 38% of the speed you're paying for."

## Quote Builder (Agent 4) — Manual Quotes

AI prepares everything for sales team:
- Customer details
- Speed test results
- Coverage status
- Pain summary
- Requirements
- Recommended package
- Risk factors
- Competitor info
- Current spend

Hands a ready-to-review quote pack to the human.

## Follow-up Machine (Agent 5)

- Day 1: Thank you
- Day 3: Coverage update
- Day 6: Educational article
- Day 10: Price comparison
- Day 15: Reminder
- Day 25: Final follow-up

## Sales Coach (Agent 6)

Analyses every lost sale:
- Price / Coverage / Delay / Competition / Wrong package / No budget

After 500 deals tells you:
- "Most losses occur because quotes take too long."
- "Business customers close 4x faster than residential."

## Dashboard (Agent 7)

Displays:
- Quote requests this week (NORTH STAR)
- Today's leads
- Hot leads
- Quotes outstanding
- Average response time
- Close rate
- Revenue pipeline
- Most requested package
- Top area
- Worst area
- Most common competitor

## Growth Engine

Continuous optimisation of:
- AI discoverability (ChatGPT, Claude, Perplexity, Gemini)
- SEO (metadata, schema, Core Web Vitals, indexing)
- Conversion rate (friction, abandonment, lead capture)
- A/B testing (headlines, CTAs, forms, share prompts)
- Viral growth (referrals, WhatsApp, LinkedIn, QR, social proof)
- Telemetry (impressions → visitors → tests → shares → leads → quotes → sales)

Loop: Observe → Measure → Analyse → Hypothesise → Experiment → Validate → Deploy → Monitor
